# Aprendizaje Automático - Semana 1

Este repositorio contiene los ejercicios prácticos de la **Semana 1** de la materia *Aprendizaje Automático*, realizados en Python utilizando la librería **NumPy**.

## 📚 Contenidos de la clase
- Conceptos previos de Machine Learning
- Aprendizaje supervisado y no supervisado
- Flujo de trabajo en ciencia de datos
- Roles dentro de un equipo de IA
- Herramientas fundamentales: Anaconda, Jupyter, NumPy, Pandas, Matplotlib, Scikit-learn

## 📝 Ejercicios incluidos
1. Crear un array de 10 elementos con ceros.  
2. Generar un array con los números del 1 al 20.  
3. Crear un array con los números pares del 2 al 50.  
5. Crear una matriz de 4x4 con valores aleatorios entre 0 y 1.  
6. Realizar la suma elemento a elemento de los arrays `a=[1,2,3]`, `b=[4,5,6]`.  
7. Multiplicar dos matrices 3x3 de enteros aleatorios entre 1 y 10.  
8. Crear un array de 15 valores aleatorios entre 1 y 100 e imprimir su máximo, mínimo y promedio.  
9. Generar una matriz 5x5 con valores del 1 al 25 y extraer su diagonal.  
10. Dada una matriz 5x5 de enteros aleatorios entre 1 y 20, calcular la suma de cada fila y columna.
